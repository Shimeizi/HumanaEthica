# 🤝 HumanaEthica

A project developed for the **Software Engineering** course.

## 📖 Overview

HumanaEthica is a volunteer management platform that connects institutions and volunteers through activities, enrollments, participations and event scheduling.

This project extends the platform by introducing **Shifts**, allowing activities to be divided into multiple time slots with participant limits and scheduling constraints.

The system follows a layered architecture with domain models, services, REST APIs, automated testing and a web-based user interface.

## ✨ Features

### Backend

* 🏢 Activity management
* 👥 Volunteer enrollments
* ✅ Participation management
* 🕒 Shift creation and scheduling
* 📋 Activity shift listing
* 🔒 Domain invariants and business rules
* 🌐 REST API services
* 🧪 Unit, component and integration testing

### Frontend

* 🖥️ Activity management interface
* 🕒 Shift management interface
* 📑 Enrollment workflows
* 👤 Volunteer interactions
* 🧪 End-to-end testing with Cypress

## 🛠️ Built With

* Java
* Spring Boot
* TypeScript
* Vue.js
* Cypress
* JUnit

## 🎯 Project Goals

The project aims to:

* Support activities composed of multiple shifts
* Enforce domain constraints through business rules
* Maintain consistency between activities, shifts, enrollments and participations
* Provide secure and validated REST services
* Ensure software quality through automated testing

## 🧪 Testing

* Web service tests
